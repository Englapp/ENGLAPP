How to Play Level Five
1. Start the Game: When you begin, you'll see a timer and a question on the screen.

2. Answer the Questions:

- There are different types of questions:
- Multiple Choice (abc): Select the correct answer from the options provided.
- Matching (match): Pair sentences with their corresponding images. Choose the correct sentence for each image.
- Ordering (order): Type the sentence in the correct order.

3. Time Limit: You have 30 seconds to answer each question. If time runs out, it will be marked as incorrect automatically.

4. Earn Coins: For each correct answer, you will earn coins. The remaining time when you answer correctly affects how many coins you earn.

5. Keep Advancing: After each question, click the "Next Question" button to continue. If you complete all the questions, you will receive a
congratulatory message and be redirected to the level selection screen.

6. Have Fun and Learn!: This game is not only entertaining but also helps you learn about colors and phrases in English.
